
************ Traducteur ******************
1/ excecuter le script makePot.sh : il construira ou écrasera le fichier modèle locale/DEVSimPy.pot
2/ se diriger dans le repertoire locale et excecuter le script make makeFrPo.sh pour creer un nouveau fichier a traduire en francais ou le script
updateFrPo pour mettre a jour le fichier fr/LC_MESSAGES/DEVSimPy.po.
3/ se diriger dans fr/LC_MESSAGE ajouter les traductions dans le fichier .po et executer le script compile.sh pour creer le fichier DEVSimPy.mo

Remarque: pour la procédure de mise à jour du fichier .mo de la langue anglaise, pas besoin de faire les modijs dans le fichiet .po
